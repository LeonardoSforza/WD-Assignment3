Sending requests
Resource HTTP Method REST API operation Description

Examples of use
Example requestes and responses for the WebShop API
Resource Request verb Description Status code
/phone-list GET Get a list of all phones in the list 200 OK
/phone-list/1 GET Get the phone in the list with ID 1 200 OK
/phone-list/999 GET Get non-existing phone with ID 999 404 Not Found
/phone-list POST Create a new phone in the list 201 Created
/phone-list/100 PATCH Update phone with ID 100 200 All Good
/phone-list/100 DELETE Delete phone with ID 100 200 All Good
